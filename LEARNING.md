## Terminal/GitHub-README refinement pass

**What we added / changed**
Refined the terminal-window profile card (generated by `ascii_profile.py`) to
read like a real terminal + GitHub README instead of a marketing landing page:
- Progress bars: 42 → 46 segments (longer) and percentages pulled ~10px left so
  the gap to each bar shrank from ~50px to ~11px. Proportions unchanged.
- Center divider extended downward (`H - TB - 54` → `H - TB - 32`, ends ~y610)
  so it balances both columns instead of stopping mid-bar.
- Removed the blinking `█` cursor from `signoff.svg`; text is now static.
- Dropped "PROBLEM SOLVER" from the subtitle → "SYSTEM ENGINEER · BUILDER".
- Typography: swapped the font stack to prefer authentic terminal fonts
  (`'SF Mono', 'Cascadia Code', 'JetBrains Mono', Menlo, Consolas, monospace`)
  and dropped the name from bold/30px to 500-weight/26px.
- Nudged the whole right column down 16px (one wrapping `<g transform>`) so it
  centers against the taller left portrait.
- Accent color: orange `#f97316` → muted bronze `#B08D57`, defined once as the
  `GOLD` constant and used only for accents (`>`, `//`, `$`, `·`, bars, stat
  keys, EMAIL outline, cursors). All body text stays white/gray. macOS window
  dots kept their real red/yellow/green (authentic chrome, not accent).
- Bumped README image cache tag `?v=31` → `?v=32`.

**Tools / concepts involved (one line each)**
- SVG (Scalable Vector Graphics): an image drawn from shapes + text, so it stays
  crisp and can animate — the whole card is one SVG built by a Python script.
- SMIL animation: the `<animate>` tags inside the SVG that fade/draw elements in.
- GitHub `<img>` sandbox: README images render in a locked mode that blocks web
  fonts, so the card can only use fonts already installed on the viewer — hence a
  font *stack*, not a single font.
- "camo" cache: GitHub proxies README images and caches them hard, so a new
  `?v=` query string is needed to force a re-fetch after changes.

**The one-sentence why**
"I refined spacing, type weight, and swapped the accent to a muted bronze so the
card reads like an engineer's terminal, not a marketing page — without touching
the layout."
